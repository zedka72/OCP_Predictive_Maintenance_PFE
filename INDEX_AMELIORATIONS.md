## ❓ FAQ Rapide

**Q : Par où commencer ?**  
A : Lire `SYNTHESE_AMELIORATIONS.txt` (5 min), puis `CHECKLIST_IMPLEMENTATION.md` (plan d'action)

**Q : Que faire si je n'ai pas les images ?**  
A : Le rapport compile sans images (avec avertissements). Les images renforcent la crédibilité mais ne sont pas obligatoires.

**Q : Peux-je compiler sans Overleaf ?**  
A : Oui ! Installer LaTeX localement et exécuter `pdflatex rapport.tex && ...` (voir `GUIDE_COMPILATION.md`)

**Q : Quel outil pour créer les images ?**  
A : Canva (facile), Draw.io (technique), PowerPoint (rapide). Peu importe tant que c'est PNG ou PDF.

**Q : Combien de temps tout ça ?**  
A : ~2-4 heures (dépend création images et expérience LaTeX)

**Q : Le rapport est sûr de compiler sans erreurs ?**  
A : Oui ! Code LaTeX validé. Si erreurs, voir `GUIDE_COMPILATION.md` section "Troubleshooting"
